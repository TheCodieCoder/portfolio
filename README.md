# Misbah Ur Rahman Portfolio

Production-ready MERN portfolio with React, Three.js, Express, MongoDB, and Gemini AI.

## Quick Start

```
cd portfolio
npm install
npm run install:all
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
npm run dev
```

Frontend: http://localhost:5173  
Backend: http://localhost:5000/api
