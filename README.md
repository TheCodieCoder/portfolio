## Overview

A clean, modern, and interactive portfolio website designed to showcase projects, skills, and personality. Built with performance and user experience in mind — featuring 3D animations via Three.js and an AI-powered assistant through Gemini AI.

> 🚧 **Currently in final stages** — minor polishing left. Deployment to Render is coming very soon.

---

## Tech Stack

| Layer      | Technology                          |
|------------|-------------------------------------|
| Frontend   | React.js, Three.js, Vite            |
| Backend    | Node.js, Express.js                 |
| Database   | MongoDB (Mongoose)                  |
| AI         | Google Gemini AI                    |
| Deployment | Render                              |
| Tooling    | Prettier, EditorConfig, VSCode      |
